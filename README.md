Objetivo: Sistema para cadstro de Cerjeva

##Funcionalidades

CRUD de Cerjeva e de cartegoria de cerveja.

##Tecnologias Utilizadas

Java, Spring Boot, Rest Assured, Spring Actuator, Docker, JPA e Postgres
